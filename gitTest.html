깃 리모트 저장소 생성 후 (이름 gitTest 로 지음) 1. 인텔리제이 로컬 작업공간에 gitTest 라는 폴더 생성 2. test 가지를 생성했고 -> git checkout test 로 작업가지를 test 가지로 변경 3. test 가지에서 파일작업을 한 뒤(gitTesth.html) 4. ★test 가지에서 git push origin test(가지이름) => origin 리모트 저장소에 test 가지를 업도르 하겠다 => 결과 gitTest 리모트 저장소에 main, test 가지 2개가 생겼고 => test 가지를 열어보면 test 가지를 업로드한 내역이 커밋되어있다!!
